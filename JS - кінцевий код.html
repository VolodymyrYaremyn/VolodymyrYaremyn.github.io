2. Your First JavaScript Program

2.1 Твоя перша програма на JavaScript

У цьому уроці ти напишеш програму, яка виведе на екран загальну кількість готівки, що зберігається в гаманці та у кишені. Але для початку, розберемось що ж таке комп’ютерна програма. Це текст, який складається з послідовності команд, що виконуються комп'ютером зверху вниз. Наприклад:

let wallet = 200;
let myLeftPocket = 50;
let totalAmount = wallet + myLeftPocket;

console.log(totalAmount);

Тут ти можеш побачити команди — рядки, які виконують певну дію та завершуються крапкою з комою ;. Перша команда оголошує змінну wallet та присвоює їй значення 200. Для цього ми використали ключове слово let. Ключові слова заздалегідь визначені мовою JavaScript та завжди пишуться маленькими літерами. Пізніше ми познайомимось і з іншими ключовими словами.

Змінна — це наче «коробка» з назвою, куди можна щось покласти. Після let ми ставимо пробіл і пишемо назву змінної, в нашому випадку це wallet. Назва має складатися з літер та пояснювати її призначення. Якщо вона складається з декількох слів, то всі слова крім першого пишуться з великої літери, наприклад, myLeftPocket. Такий стиль написання називається camelCase.

Після назви змінної пишемо оператор присвоєння =, який дозволяє присвоїти змінній певне значення, а простими словами — «покласти щось до коробки». Зараз ми зберігаємо число 200 у змінній wallet та число 50 у змінній myLeftPocket. Навколо кожного оператора потрібно ставити пробіли, тоді код легше читається. Також до змінної можна зберегти результат обчислення, наприклад:

let totalAmount = wallet + myLeftPocket;

Оператор +, так само як і в математиці, обчислює суму двох чисел.

Щоб вивести результат на екран, потрібно написати console.log, і далі в дужках вказати назву змінної:

console.log(totalAmount);

Як запустити завдання на перевірку? Перейди на вкладку Практика та натисни червону кнопку внизу екрана — Запустити. На панелі результатів праворуч розгорни Логи. Ти побачиш число 250 — значення змінної totalAmount — загальну кількість грошей. Задачу розв’язано ✅ За потреби можна змінити кількість грошей у гаманці та у кишені, а програма все одно буде рахувати загальну кількість.

А тепер переходь до практики та напиши свою першу програму на JavaScript самостійно 😎

3. Main Concepts

3.1 Основні поняття

У цьому топіку розглянемо основні поняття мови JavaScript, а саме: типи даних, ключові слова та коментарі ⬇️

3.2 Типи даних

Числа (number), які можуть бути цілими та дробовими:

const age = 37;
const x = 0.25;

Рядки (string), тобто послідовності символів. Зверни увагу на те, що всі рядки пишуться в лапках:

const name = 'John';
const message = 'Hello, world!';

💡 Рядки можна записувати в 'одинарних' або "подвійних" лапках. Який тип лапок обрати? Зазвичай це залежить від особливостей конкретного проєкту та внутрішніх правил команди, з якою ти будеш працювати. Проте дуже важливо дотримуватися одного типу лапок в усьому проєкті, щоб зберегти зрозумілість коду.

Булевий тип (boolean), який може приймати лише значення true або false (правда чи брехня):

const isMarried = true;
const hasCat = false;

3.3 Ключові слова

У мові JavaScript існують ключові слова. Вони «вбудовані» в програму, мають спеціальне призначення та не можуть бути використані, наприклад, як назви змінних.

Прикладами ключових слів є let та const, які використовуються для оголошення (створення) змінних. Зверни увагу на їх відмінність: значення змінної, оголошеної за допомогою let, можна міняти після її оголошення, а за допомогою const — не можна. Розглянемо приклад:

let x = 5;
x = 10; // можна змінити значення `x` з 5 на 10

const y = 5;
y = 10; // тут буде помилка, бо неможливо змінити значення `y` після оголошення

3.4 Коментарі

Іноді у коді потрібно пояснити деякі дії. Для цього використовуються коментарі — це такі тексти, які не виконуються комп’ютером, а просто допомагають програмісту краще зрозуміти логіку програми. Коментарі бувають:

Однорядкові, які починаються з двох слешів //:
// надрукуємо в консолі фразу 'Hello, world!'
console.log('Hello, world!');

Багаторядкові, які позначаються символами /* на початку та */ у кінці:
/* це багаторядковий коментар
цей рядок не виконається
і цей теж */

console.log('Hello, world!');

4. Numbers

4.1 Числа

У JavaScript всі числа належать до типу number. Вони можуть містити як цілі, так і дробові значення.

Розглянемо, які операції ми можемо виконувати з числами, а також їх пріоритет.

4.2 Операції

Думаю ти ще пам'ятаєш з уроків математики, що числа можна додавати, віднімати, множити, ділити, підносити до ступеня та брати остачу від ділення. Розглянемо приклади:

let addition = 10 + 5; // додавання
console.log(addition); // 15

let subtraction = 10 — 5; // віднімання
console.log(subtraction); // 5

let multiplication = 10 * 5; // множення
console.log(multiplication); // 50

let division = 10 / 5; // ділення
console.log(division); // 2

let exponentiation = 10 ** 5; // піднесення до степеня
console.log(exponentiation); // 100000

let remainder = 16 % 7; // остача від ділення
console.log(remainder); // 2 — віднімаємо 7 поки результат не стане меншим за 7

За допомогою оператора % можна отримати останню цифру числа, наприклад:

let number = 345 % 10;
console.log(number); // 5

Також можна перевірити, чи є число парним:

let number = 345 % 2; console.log(number); // залишок 1 — число непарне

4.3 Пріоритет

Зазвичай операції виконуються по черзі зліва направо. Але множення та ділення мають вищий пріоритет, тому виконуються раніше, ніж додавання та віднімання. Наприклад:

console.log(
5 + 1 * 10 // 15
);

Щоб встановити правильну послідовність обчислень, потрібно скористатися круглими дужками (). Тоді спочатку виконається дія в дужках, а потім усі інші:

console.log(
(5 + 1) * 10 // 60
);

Якщо у виразі буде декілька пар дужок, то вони виконуються зліва направо.

5. Strings

5.1 Рядки

Рядок (string) — це послідовність символів, таких як літери, цифри, спеціальні символи тощо. Рядки використовуються для представлення текстової інформації у програмах.

💡 Рядки потрібно писати у лапках. Лапки на початку та в кінці рядка повинні бути однаковими.

5.2 Довжина рядка

Дізнатися довжину рядка (кількість символів у ньому) можна за допомогою властивості length:

const name = 'John';

console.log(name.length); // 4

5.3 Символ у рядку

Отримати окремий символ у рядку можна за його індексом. Індекси в рядку починаються з 0 (не з 1). Наприклад:

const name = 'John';

console.log(
name[0], // 'J'
name[1], // 'o'
name[2], // 'h'
);

Також в квадратних дужках можна написати вираз і обчислити індекс. Наприклад, індекс останнього символу на 1 менше за довжину рядка:

const name = 'John';
const lastIndex = name.length - 1;

console.log(
name[lastIndex], // 'n'
name[name.length - 1], // 'n' - обчислення прямо в []
);

5.4 Конкатенація

Конкатенація — це обʼєднання («склеювання») рядків за допомогою оператора +. У цьому контексті оператор + виконує саме об’єднання рядків, а не додавання чисел:

const name = 'John';
const lastName = 'Stewart';

const initials = name[0] + lastName[0];
console.log(initials);// 'JS'

const fullName = 'Mr.' + name + ' ' + lastName;
console.log(fullName);// 'Mr. John Stewart'

Конкатенувати можна не тільки рядки, а й рядки з числами. Зверни увагу на те, що числа в лапках є рядками:

console.log(
'12' + 34, // '1234'
12 + '34', // '1234'
);

5.5 Інтерполяція

Інтерполяція — це вставка значень у рядок. Для здійснення інтерполяції використовується синтаксис ${...}:

const name = 'John';
const lastName = 'Stewart';

const fullName = `Mr. ${name} ${lastName}`;

console.log(fullName); // 'Mr. John Stewart'

Розглянемо ще один приклад, де покажемо різницю між конкатенацією та інтерполяцією рядків:

const name = 'John';
const age = 30;

// конкатенація
const concatenationGreeting = 'Hello! My name is ' + name + ' and I am ' + age + '.';
console.log(concatenationGreeting); // 'Hello! My name is John and I am 30.'

// інтерполяція
const interpolationGreeting = `Hello! My name is ${name} and I am ${age}.`;
console.log(interpolationGreeting); // 'Hello! My name is John and I am 30.'

У більшості випадків інтерполяція більш наочна, тому її використовують набагато частіше.

6. Boolean

6.1 Булевий тип

Булевий тип використовується для значень виду «так, вірно» або «ні, не вірно»: true (істина) та false (хиба) відповідно:

const isAdult = true;
const hasEnoughMoney = false;

6.2 Порівняння чисел

Булеві значення також можна отримати у результаті порівняння, наприклад, чисел. Для їх порівняння у JavaScript використовуються оператори з уроків математики:

> — більше;
< — менше;
>= — більше або дорівнює;
<= — менше або дорівнює.
Також для порівнянь можна застосовувати наступні оператори:

=== — строга рівність;
!== — строга нерівність.
💡 Два попередніх оператори не слід плутати з оператором =, який використовується для присвоєння значень змінним.

А тепер розглянемо приклади:

const a = 2;

console.log(
a === 2, // true
a !== 2, // false
a > 2, // false
a < 2, // false
a >= 2, // true
a <= 2, // true
);

Ще один приклад із відео:

const age = 25;

let isAdult = age >= 18;
console.log (isAdult); // true

const isNotAdult = age < 18;
console.log (isNotAdult); // false

6.3 Оператор заперечення (!)

Щоб отримати протилежне значення змінної, можна використати оператор логічного заперечення !:

const age = 25;
let isAdult = age >= 18;

console.log(isAdult); // true
console.log(!isAdult); // false

6.4 Логічне І (&&)

Якщо потрібно перевірити, що дві умови вірні одночасно, можна використати оператор && — логічне І. Його результатом буде true, якщо обидві умови є вірними (true):

let cash = 50;
let price = 40;
let hasEnoughCash = cash >= price;

let age = 25;
let isAdult = age >= 18;

let canBuy = isAdult && hasEnoughCash;

console.log(canBuy); // true

6.5 Логічне АБО (||)

Якщо потрібно перевірити, що хоча б одна з умов є вірною, можна використати оператор || — логічне АБО. Його результатом буде true, якщо хоча б одна з умов є вірною (true):

let cash = 50;
let creditCard = 20;
let price = 40;

let hasEnoughCash = cash >= price;
let hasEnoughCredit = creditCard >= price;

let canBuy = hasEnoughCash || hasEnoughCredit;
console.log(canBuy); // true

6.6 Комбінації клавіш з уроку

Перейменувати змінну: F2.
Скопіювати рядок вгору: Shift + Alt + ↑ (на Windows) або Shift + Option + ↑ (на macOS).
Скопіювати рядок вниз: Shift + Alt + ↓ (на Windows) або Shift + Option + ↓ (на macOS).

7. Functions

7.1 Функції

Функція — це блок коду, який виконує конкретну задачу. Функції є основними «будівельними блоками» програми, і вони дозволяють використовувати один і той самий код декілька разів з різними даними, уникаючи дублювання.

7.2 Оголошення функції (function declaration)

У JavaScript існує декілька способів створення функції. Одним із них є оголошення (тобто створення) функції за допомогою ключового слова function, подібно до того, як let та const оголошують змінні. Ось приклад функції, яка обчислює площу прямокутника, та друкує її в консолі:

function calculateArea(length, width) {
let area = length * width;

console.log(area);
}

Тут:

  • function — ключове слово, яке оголошує функцію;
  • calculateArea — ім'я функції, за яким обовʼязково йдуть круглі дужки ();
  • length та width — параметри функції, це змінні, які отримають значення під час використання функції;
  • усі команди, що записані у фігурних дужках {}, називаються тілом функції.
Функції можуть мати один параметр, кілька параметрів або не мати параметрів взагалі.

7.3 Виклик функції

Щоб програма виконала команди, які записані у тілі функції, цю функцію потрібно викликати. Для виклику функції потрібно написати її ім'я та в круглих дужках вказати аргументи — конкретні значення параметрів функції. Викличемо функцію calculateArea з аргументами 5 та 7:

function calculateArea(length, width) {
let area = length * width;

console.log(area);
}

calculateArea (5, 7); // у консолі надрукується 35

Зверни увагу: при обчисленні площі значення 5 потрапить до параметра length, значення 7 — до параметра width.

Інколи один із параметрів має заздалегідь визначене значення або значення за замовчуванням. Наприклад, якщо ми обчислюємо площу рулона тканини, що має стандартну ширину 1.5 метри:

function calculateArea(length, width = 1.5) {
  let area = length * width;

console.log(area);
}

calculateArea(5); // значення `width` буде 1.5, тому результат обчислень — 7.5

Значення за замовчуванням схожі на стандартні налаштування телефона, які встановлює виробник. Вони визначені завчасно, проте ти завжди можеш змінити їх, якщо потрібно. Так і параметри функції за замовчуванням можуть змінюватись, якщо ти передаш у них нові значення при виклику функції:

function calculateArea(length, width = 1.5) {
  let area = length * width;

console.log(area);
}

calculateArea (5); // значення `width` буде 1.5, бо ми не передали іншого, результат обчислень — 7.5
calculateArea (5, 7); // для обчислень використовуються вказані значення, результат обчислень — 35

13. Working With Arrays

13.1 Робота з масивами

У реальному житті ми дуже часто стикаємося з великою кількістю однотипних даних: слова в словнику, оцінки в журналі, товари в магазині, книги на полиці тощо. Ми вже раніше розглянули, що у JavaScript для роботи з такими даними використовуються масиви — структури даних, які дозволяють зберігати декілька значень в одній змінній. У цьому уроці ти дізнаєшся, як використовувати їх на практиці.

13.2 Сума чисел з масиву

Однією із розповсюджених операцій при роботі з масивами є знаходження суми чисел. Наприклад, якщо масив містить ціни на товари в кошику, ми можемо порахувати загальну вартість. Порахуємо суму чисел з масиву за допомогою циклу for:

const prices = [10, 20, 30, 40, 50];
const totalPrice = getSum(prices);

console.log(totalPrice);

function getSum(values) {
    let sum = 0;

    for (let i = 0; i < values.length; i++) {
      sum += values[i];
    }

   return sum;
}

Як працює цей код?

1. Спочатку ми викликаємо функцію getSum і передаємо їй наш масив prices.
2. Усередині функції ми оголошуємо змінну sum, де буде накопичуватися сума чисел.
3. За допомогою циклу for ми перебираємо елементи масиву та додаємо поточний values[i] до суми.
4. Після завершення циклу ми повертаємо значення змінної sum.

Те саме можна зробити за допомогою циклу for of:
const prices = [10, 20, 30, 40, 50];
const totalPrice = getSum(prices);

console.log(totalPrice);

function getSum(values) {
  let sum = 0;

  for (const value of values) {
    sum += value;
  }

  return sum;
}

13.3 Середнє значення

Щоб знайти середнє значення чисел в масиві, потрібно обчислити суму всіх елементів та розділити її на кількість елементів. Наприклад:

function getAverage(values) {
  let sum = 0;

  for (const value of values) {
    sum += value;
}

  return sum / values.length;
}

const prices = [10, 20, 30, 40, 50];
const averagePrice = getAverage(prices);

console.log(averagePrice);
Але якщо масив порожній, то доведеться ділити на 0, що призведе до помилки обчислень NaN. Такий випадок потрібно обробити окремо на початку функції — якщо в масиві немає елементів, то повертаємо 0:
function getAverage(values) {
  if (values.length === 0) {
    return 0;
  }

  // далі той саме код, що й раніше
}

13.4 Мінімальне та максимальне значення

Мінімальне та максимальне значення Іноді нам потрібно знайти найменше або найбільше значення в масиві. Для прикладу знайдемо найменше значення:

function getMin(values) {
  if (values.length === 0) {
    return 0;
  }

  let min = values[0];

  for (const value of values) {
    if (value < min) {
      min = value;
    }
  }

  return min;
}

const prices = [10, 20, 30, 40, 50];
const minPrice = getMin(prices);

console.log(minPrice);

13.5 Робота з масивами рядків

Виконання операцій з масивами рядків може бути корисним, коли ми працюємо зі списками імен, товарів або будь-якою іншою текстовою інформацією. Перетворення рядка у масив (split) Метод split«розрізає» рядок на окремі частини за вказаним роздільником і повертає масив з цими частинами. Роздільник вказується як аргумент методу split. Якщо роздільником є порожній рядок, то отримаємо масив окремих символів: const text = 'one two three four five'; const chars = text.split(''); concole.log(chars); // ['o', 'n', 'e', ...] Якщо роздільником виступає один пробіл ' ', то рядок розбивається на окремі слова: const text = 'one two three four five'; const words = text.split(' '); concole.log(words); // ['one', 'two', 'three', ...] Роздільником може виступати не один символ, а й цілий рядок: const text = 'one two three four five'; const words = text.split(' f'); concole.log(words); // ['one two three', 'our', 'ive'] Фільтрування елементів масиву Іноді потрібно створити новий масив, де будуть ті елементи вихідного масиву, які задовольняють умові. У такому випадку можна додати умову в цикл. Розглянемо приклад, де ми відфільтровуємо короткі слова та складаємо їх у новий масив: function getShortWords(words, maxLength) { let results = []; for (const word of words) { if (word.length <= maxLength) { results.push(word); } } return results; } const text = 'one two three four five'; const words = text.split(' '); console.log( getShortWords(words, 4) );
